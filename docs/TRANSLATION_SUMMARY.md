# Resumen de Traducción al Español

Este documento lista todas las traducciones de la interfaz de usuario de inglés a español.

## Traducciones Generales

### Botones y Acciones
- **Add** → **Agregar**
- **Edit** → **Editar**
- **Delete** → **Eliminar**
- **Update** → **Actualizar**
- **Create** → **Crear**
- **Cancel** → **Cancelar**
- **Save** / **Saving...** → **Guardar** / **Guardando...**
- **Actions** → **Acciones**
- **Copy** → **Copiar**

### Formularios
- **Name** → **Nombre**
- **Full Name** → **Nombre Completo**
- **Position** → **Puesto**
- **Department** → **Departamento**
- **Password** → **Contraseña**
- **Serial Number** → **Número de Serie**
- **Assigned To** → **Asignado a**
- **Select employee** → **Seleccionar empleado**
- **Unassigned** → **Sin asignar**

### Departamentos
- **Logistics** → **Logística**
- **Finance** → **Finanzas**

### Empleados
- **Employee** → **Empleado**
- **Add Employee** → **Agregar Empleado**
- **Edit Employee** → **Editar Empleado**
- **Delete employee** → **Eliminar empleado**
- **Copy employee ID** → **Copiar ID del empleado**

### Laptops
- **Laptop** → **Laptop**
- **Add Laptop** → **Agregar Laptop**
- **Edit Laptop** → **Editar Laptop**
- **Delete laptop** → **Eliminar laptop**
- **Cellphone** (en contexto de laptop) → **Celular**

### Celulares
- **Cellphone** → **Celular**
- **Add Cellphone** → **Agregar Celular**
- **Edit Cellphone** → **Editar Celular**
- **Delete cellphone** → **Eliminar celular**
- **Phone** → **Teléfono**
- **IMEI** → **IMEI** (se mantiene)

### Emails
- **Email** → **Correo** / **Email**
- **Add Email** → **Agregar Email**
- **Edit Email** → **Editar Email**
- **Delete email** → **Eliminar email**
- **Email Type** → **Tipo de Email**
- **Email Address** → **Dirección de Email**
- **Username** → **Usuario**

### Tipos de Email
- **Administrative** → **Administrativo**
- **Gmail** → **Gmail** (se mantiene)
- **Hotmail** → **Hotmail** (se mantiene)
- **iCloud** → **iCloud** (se mantiene)
- **Hosting** → **Hosting** (se mantiene)

### Mensajes de Confirmación
- **Are you sure you want to delete {name}?** → **¿Estás seguro de eliminar a {name}?**
- **Update employee information** → **Actualizar información del empleado**
- **Add a new employee to the system** → **Agregar un nuevo empleado al sistema**
